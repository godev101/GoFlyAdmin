# 002前端项目安装.md

```shell
## 
nvm install v20.11.0
nvm alias default v20.11.0
## 
## 初始化前端依赖
npm install --registry=https://registry.npmmirror.com
or
npx yarn install
## 运行前端项目
npm run serve
or
npx yarn serve
## 
## 
# 后端打包
CGO_ENABLED=0 GOOS=linux GOARCH=amd64 go build -v .
## copy gofly二进制文件 和 resource 文件夹到Linux服务器上
## 
## 前端打包
npm run build or
npx yarn build

## 
##
```
